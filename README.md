# 🎮 Simon Says Game

A simple memory game built with HTML, CSS, and JavaScript.

## 🕹️ How to Play
- Watch the color pattern shown.
- Repeat the pattern by clicking the buttons.
- One wrong move ends the game!

## 📁 Files
- `index.html` – Structure
- `style.css` – Styling
- `app.js` – Game logic
